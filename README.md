# 📝 Blog REST API

A simple RESTful API for managing blog posts, built with **Node.js**, **Express.js**, and **MongoDB** using **Mongoose**.

---

## 🚀 Features

- Create a blog post
- Retrieve all blog posts
- Retrieve a single blog post by ID
- Update a blog post
- Delete a blog post

---

## 🧱 Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- dotenv

---

## 📁 Folder Structure

blog-api/
